Discussion Threads from the Forums of 60 Coursera Massive Open Online Courses (MOOCs)

For CMS.595 class, I created a data visualization of the discussion threads from the forums of 60 Coursera MOOCs in order to get some insights of the culture of the online classes. On the left, you will see a bubble chart where each bubble represents a MOOC class. The color and the size of each bubble refer to the subject and the number of users of the corresponding class. The right side of this page shows a donut chart that shows the topics of all the threads in a MOOC's forum. Once you select a course, the donut chart will change to reveal the thread topics in that class. The bar graph shows the number of posts by various user types including students, anonymous, instructors, and Couresa staff. After playing around with this data visualization, I drew some inferences from the data.

The dataset that this website visualized is from this GitHub repo, which was used in the paper Language Independent Analysis and Classification of Discussion Threads in Coursera MOOC Forums
